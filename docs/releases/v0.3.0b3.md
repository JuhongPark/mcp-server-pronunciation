# v0.3.0b3 Public Beta

Public beta notice: this is an early beta and is still under active
development. It may contain bugs, runtime errors, inaccurate transcripts,
inaccurate pronunciation feedback, or platform-specific recording issues. Use
it for experimentation and language-learning practice only, and review outputs
carefully before relying on them.

## Highlights

- Public package metadata and README positioning now describe the project as a
  general local English voice and pronunciation coach.
- Learner-profile hints remain available. The current rule pack includes
  Korean-L1 pronunciation-pattern hints, and contributions for additional L1
  profiles are welcome.
- User-facing assessment reports now label language-background rules as
  learner-profile hints.
- Includes the v0.3 assessment pipeline with word alignment, phoneme feedback,
  optional wav2vec2 forced alignment, drills, and prosody checks.

## Install The Beta

```bash
uvx mcp-server-pronunciation@0.3.0b3
```

For pip users:

```bash
pip install --pre mcp-server-pronunciation
```

Run the preflight before wiring it into an MCP client:

```bash
mcp-server-pronunciation doctor
```

# v0.3.0b2 Public Beta

Public beta notice: this is an early beta and is still under active
development. It may contain bugs, runtime errors, inaccurate transcripts,
inaccurate pronunciation feedback, or platform-specific recording issues. Use
it for experimentation and language-learning practice only, and review outputs
carefully before relying on them.

## Highlights

- Reworked pronunciation assessment pipeline with word alignment, phoneme
  feedback, Korean-L1 pattern detection, and prosody checks.
- Optional wav2vec2 forced alignment through the `[phoneme]` extra.
- Public roadmap, testing methodology, benchmark adapter scaffold, and score
  metric helpers.
- MCP Registry metadata fixes for GitHub OIDC namespace casing and description
  length validation.

## Install The Beta

```bash
uvx mcp-server-pronunciation@0.3.0b2
```

For pip users:

```bash
pip install --pre mcp-server-pronunciation
```

Run the preflight before wiring it into an MCP client:

```bash
mcp-server-pronunciation doctor
```
